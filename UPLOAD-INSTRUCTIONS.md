# How to publish this portfolio on GitHub

## Option 1 — Upload through the browser

1. Open your `Me` repository.
2. Select **Add file** → **Upload files**.
3. Extract the ZIP file on your computer.
4. Drag all contents inside the extracted `Me` folder into GitHub.
5. Use the commit message: `Build executive transformation portfolio`.
6. Select **Commit changes**.

## Important profile note

A repository called `Me` will work as a normal portfolio repository.
